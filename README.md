# Base Jurema — Escola & Passeios de Canoa Havaiana (Vitória - ES)

Web Application moderna, interativa e de alta conversão para **Base Jurema** na Curva da Jurema, Vitória / ES.

## 🚀 Tecnologias Utilizadas

- **Framework**: Next.js 15.5+ (App Router) + React 19 + TypeScript
- **Estilo**: Tailwind CSS v4 + Lucide React + Identidade Verde Lima Flúor (`#CFF726`)
- **Implantado em**: Vercel

## 🌊 Funcionalidades

- **Motor de Reservas Interativo**: Escolha de passeio, data, horário, remadores e adicionais (fotos de drone, café da manhã) com cálculo automático em R$ e envio direto ao WhatsApp (`+55 27 99931-0057`).
- **Widget de Clima & Marea em Tempo Real**: `/api/weather` para a Curva da Jurema.
- **Roteiro Marítimo Interativo**: Curva da Jurema ➔ Ilha do Frade ➔ Ilha do Boi ➔ Terceira Ponte.
- **Portal Escolinha & Wellhub (Gympass)**: Guia de planos e check-in.
- **Ubicación en Google Maps**: Embed oficial do local.

## 📦 Como rodar o projeto localmente

```bash
git clone https://github.com/melcheikh/basejurema-web.git
cd basejurema-web
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador.
